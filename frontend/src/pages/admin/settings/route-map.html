Key Policy Rules
  • Finance & Logistics: Can only view their own profile — no user management authority
  • Finance dispute resolution: Blocked from resolving their own disputes (conflict of interest)
  • Manager: Can only update subordinate employees (manager_id match)
  • Super Admin: Bypasses all checks via before() hook in policies
  • Driver: Like vendor — portal access only, zero admin authority

Route Protection Rules
  • Public routes: No authentication required — accessible to anyone
  • Customer routes: Require auth:sanctum + role:customer
  • Admin routes: Require auth:sanctum + any of: admin, super_admin, manager, finance, logistics, sales_rep
  • Super admin routes: Additional role:super_admin middleware on destructive actions
  • Finance routes: Isolated under /admin/finance/* with role:finance gate
  • Careers portal: Separate auth system — applicant model, not customer model

🛡 Policy Gate Architecture
How Laravel policies enforce access control across models and actions 
flowchart TB
  subgraph USER_POLICY["UserPolicy"]
    direction TB
    UP1["Hierarchy: super_admin(1) → admin(2) → manager(3) → finance(4) → logistics(5) → sales_rep(6) → customer(7) → vendor(8) → driver(9)"]
    UP2["outranks(): lower level = higher authority"]
    UP3["viewAny(): isAdmin()"]
    UP4["view(): outranks() OR self (finance/logistics only)"]
    UP5["create(): super_admin, admin, manager only"]
    UP6["update(): outranks() OR self (finance/logistics)"]
    UP7["delete/restore(): outranks()"]
    UP8["forceDelete(): super_admin only"]
    UP9["canAssignRole(): level(actor) < level(role)"]
  end

  subgraph PROJECT_POLICY["ProjectPolicy"]
    direction TB
    PP1["isStaff(): super_admin, admin, manager, sales_rep"]
    PP2["participant(): admin_user_id OR customer_id match"]
    PP3["view(): super_admin OR staffGlobal OR participant"]
    PP4["update(): admin_owner/admin_manager OR owner_admin_id OR customer_owner"]
    PP5["delete(): super_admin (any) OR admin (must be participant)"]
    PP6["manageParticipants(): admin_owner/admin_manager OR customer_owner"]
    PP7["comment(): participant.can_comment = 1 (customer_viewer ✅)"]
    PP8["viewFinance(): super_admin/admin OR can_view_finance = 1"]
    PP9["manageLinks(): admin only (customers false)"]
    PP10["manageWork(): staff OR customer_owner/editor"]
  end

  subgraph PAYMENT_POLICY["PaymentPolicy"]
    direction TB
    PAY1["viewAny(): super_admin, admin, manager, finance, sales_rep, logistics"]
    PAY2["view(): finance sees own initiated only; admin/super_admin see all"]
    PAY3["create(): finance, admin, super_admin"]
    PAY4["update(): confirmed/refunded = immutable; finance sees own only"]
    PAY5["cancel(): pending only; finance sees own; admin/super_admin see all"]
    PAY6["retry(): failed/cancelled only; finance/admin/super_admin"]
    PAY7["raiseDispute(): confirmed + no existing dispute; finance/admin/super_admin"]
    PAY8["resolveDispute(): raised/investigating; finance BLOCKED from own disputes"]
    PAY9["delete(): NEVER — payment records are immutable"]
  end

  subgraph LOYALTY_POLICY["LoyaltyPolicy"]
    direction TB
    LP1["before(): super_admin bypasses all"]
    LP2["viewAny(): isAdmin()"]
    LP3["view(): admin (any) OR customer (self only)"]
    LP4["grantPoints(): super_admin, admin, manager, finance, sales_rep"]
    LP5["deductPoints(): super_admin, admin, manager, finance (NOT sales_rep)"]
    LP6["grantCredit/deductCredit(): super_admin, admin, manager, finance"]
    LP7["redeem(): admin (any) OR customer (self)"]
    LP8["configureSettings(): super_admin, admin only"]
    LP9["export(): super_admin, admin, manager, finance"]
  end

  subgraph EMPLOYEE_POLICY["EmployeePolicy"]
    direction TB
    EP1["viewAny(): super_admin, admin, manager"]
    EP2["view(): super_admin, admin, manager OR self"]
    EP3["create(): super_admin, admin only"]
    EP4["update(): super_admin (any) / admin (any) / manager (subordinates) / self"]
    EP5["delete/restore(): super_admin, admin only"]
    EP6["forceDelete(): super_admin only"]
    EP7["manageSensitiveData(): super_admin, admin only"]
    EP8["manageStatus(): super_admin, admin only"]
  end

  subgraph AI_KEY_POLICY["AiProviderKeyPolicy"]
    direction TB
